_[mnml]_ _[Alfred]_ [theme]
===========================
A minimal UI theme for [Alfred].

![preview]

[Install]
---------
**NOTE** that custom [theme]s require the paid _[Powerpack]_ upgrade to Alfred.

See the [release] page or just click [install] now.
